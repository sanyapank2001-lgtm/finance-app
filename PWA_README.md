# Контур PWA

Приложение готово к публикации как статический PWA: backend и сборка не требуются.

## Публикация

Разместите содержимое папки на любом статическом HTTPS-хостинге. Service worker не работает при открытии `index.html` напрямую с диска; нужен HTTPS-адрес (или `localhost` для разработки).

## Установка на iPhone

1. Откройте опубликованный HTTPS-адрес в Safari.
2. Нажмите «Поделиться».
3. Выберите «На экран Домой».
4. Подтвердите кнопкой «Добавить».

После первого успешного открытия приложение запускается с домашнего экрана в отдельном окне и продолжает работать без сети. Финансовые данные хранятся локально на устройстве в `localStorage`.
